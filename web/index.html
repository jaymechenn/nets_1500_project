Six Degrees of Wikipedia DEMO

Type two articles and the BFS engine finds the shortest hyperlink path between them. This demo runs in your browser on a 16-article test graph; the real project is the Java application in this folder, which runs the identical algorithm over Stanford SNAP's 1.79 million-article Wikipedia graph.

Click Find shortest path to run BFS.
About this demo. The same BFS implemented here in JavaScript is what the Java project (src/main/java/wikipath/PathFinder.java) runs at scale. To run the real thing on the full SNAP graph:
  ./scripts/download-data.sh
  java -Xmx2g -cp target/classes wikipath.Main path "Kanye West" "Quantum mechanics"
